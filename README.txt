Portfolio Website for Harvee Agravante
======================================

Folder Structure
- index.html Main HTML file.
- cssstyles.css Styling for the website.
- jsscript.js JavaScript animations and interactivity.
- images Images used in the website (e.g., profile picture).
- docs Supporting documents (e.g., CV).

Instructions
1. Open the `index.html` file in a browser to view the portfolio.
2. Update the content as needed to reflect new experience or skills.
3. Modify styles in `cssstyles.css` for additional design customizations.
